
Драйвер для VMware SVGA II.

Драйвер состоит из двух частей: загрузчика vmwgfx и драйвера ядра vmwgfx.dll.
Оба файла обязательно должны находиться в одной папке.

Запуск драйвера: <путь>/vmwgfx <командная строка>

Параметры командной строки:

 -l <полный путь к лог-файлу>
    Файл должен распологаться на устройстве доступеном для записи.
    по умолчанию /tmp1/1/drivers/vmwgfx.log 

Для запуска из autorun.dat командная строка не должна содержать пробелов. 

Пример:
/RD/1/DRIVERS/vmwgfx -l/hd0/2/vmwgfx.log 1    # 


Особенности реализации.

Нет поддержки аппаратных курсоров.

vmwgfx предназначен только для загрузки драйвера.
Для смены видеорежима в процессе работы используйте vmode или vmode <видеорежим> 
Например vmode 1152x864х75
