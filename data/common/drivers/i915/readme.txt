
Поддерживаются все PCI Express видеоядра Intel от i915 до Skylake.

Драйвер состоит из двух частей: загрузчика i915 и драйвера ядра i915.dll.
Оба файла обязательно должны находиться в одной папке.

Запуск драйвера: <путь>/i915 <командная строка>

Параметры командной строки:
 -l
 --log 	<полный путь к лог-файлу>
    	Файл должен распологаться на устройстве доступеном для записи.
    	по умолчанию /rd/1/drivers/i915.log 
 --fbc 	<-1,0,1> включает сжатие видеобуфера. Снижает нагрузку на
	системную шину и уменьшает потребление энергии. 
 --rc6 	<-1,0-7> включает режимы экономии энергии C-state 6
 -m
 --m <WxHxHz> устанавливает видеорежим на первом доступном коннекторе.

 -v
 --video <CONNECTOR>:<xres>x<yres>[M][R][-<bpp>][@<refresh>][i][m][eDd]
	устанавливает режим для коннектора CONNECTOR

Для запуска из autorun.dat командная строка не должна содержать пробелов. 

Пример:
/RD/1/DRIVERS/I915 -l/hd0/2/i915.log 1    # 

Дополнительные параметры командной строки.

 --list-connectors выводит список активных коннекторов
 --list-connector-modes <CONNECTOR>
	выводит список доступных видеорежимов для 
	коннектора CONNECTOR 
Пример:
получаем список подключенных коннекторов
i915 --list-connectors

получаем список доступных видеорежимов
i915 --list-connector-modes HDMI-A-1

устанавливаем видеорежим 1600x900 60Hz на коннекторе HDMI-A-1
i915 -v HDMI-A-1:1600x900

устанавливаем видеорежим 1280x1024 75Hz на коннекторе VGA-1
i915 -v VGA-1:1280x1024@75

Особенности реализации.

После загрузки драйвера курсор прыгает в центр экрана до первого
перемещения мышки.
