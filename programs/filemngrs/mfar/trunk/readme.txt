---[ ATTENTION ]---------------------------------------------------------------
This MFAR version is far from being RELEASE!
So, use it carefully, paying special attention to the new features.

---[ ABOUT ]-------------------------------------------------------------------
MFAR was intended as SYSTREE-alternative & at the same time as file-manager
with well-known & handy interface for MenuetOS. It'll be look like FAR
(farmanager.com) which i really like BTW.

---[ FEATURES ]----------------------------------------------------------------
 - displaying RD (left) & HD (right) file structure
 - color highlighting for files (depending on their extensions)
 - sotring (by name or extension)
 - selected file info: date, size
 - MeOS apps execution

---[ TODO ]--------------------------------------------------------------------
 - copying (moving), deleting files
 - disk selection
 - other sort types
 - not only apps execution
 - ...

---[ HOTKEYS ]-----------------------------------------------------------------
 Up, Down, Left, Right, Home, End - if you worked in any DOS-based file-
   manager (NC, VC, DN, ...), even a few minutes, you may know that this
   keys direct the caret (light pen)
 F5 - copy !FILE! from one panel to another (in development)
 F10 - exit program
 Ctrl+F3 - sort by name
 Ctrl+F4 - sort by extension

---[ Внимание ]----------------------------------------------------------------
Эту версию MFAR никак нельзя считать релизом!
Программа находится в разработке и я очень сожалею о том, что не в силах
пока что представить Вам полноценный файл-менеджер, хотя бы приблизительно
сравнимый по функциональным возможностям с аналогичными проектами под *nix
и *dows.

---[ О программе ]-------------------------------------------------------------
MFAR был задуман как альтернатива SYSTREE и одновременно как файл-менеджер
со знакомым и удобным интерфейсом под MenuetOS. Из названия можно догадаться,
что внешний вид программы будет не особо отличаться от FAR'а (farmanager.com),
очень, кстати, мною любимого и уважаемого проекта.

---[ Что сделано ]-------------------------------------------------------------
 - отображение файловой структуры образа (слева) и ЖД (справа)
 - подсветка имен файлов разными цветами (в зависимости от формата)
 - сортировка (по имени или расширению)
 - размер и дата создания (изменения) для выделенного файла
 - запуск исполняемых файлов

---[ Что хотелось бы сделать ]-------------------------------------------------
 - копирование (перемещение), удаление файлов
 - выбор активного диска для панелей
 - другие виды сортировок
 - запуск не только исполняемых файлов
 - ...

---[ Сочетания клавиш ]--------------------------------------------------------
 Up, Down, Left, Right, Home, End - кто хотя бы раз работал в ДОСовском
   файл-менеджере (NC, VC, DN, ...) знает, что с помощью этих клавиш можно
   управлять положением курсора (светового пера)
 F5 - копировать файл из одной панели в другую (в разработке)
 F10 - выход из программы
 Ctrl+F3 - сортировать по имени
 Ctrl+F4 - сортировать по расширению
