Как преобразовать иконки в подходящие для Notify 3 (из GIMP):
    1) объеденяем все слои (если их много)
    2) устанавливаем вторичный цвет в #222222
    3) Слой -> Прозрачность -> Добавить альфа-канал
    4) Слой -> Прозрачность -> Удалить альфа-канал
    5) Инструменты -> Выделение -> Выделение по цвету
    6) Ставим порог на 0 и выделяем цвет фона (#222222)
    7) Удаляем выделение и сохраняем
