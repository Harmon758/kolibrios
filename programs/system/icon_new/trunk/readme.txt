Для использования этой версии ICON необходимо заменить icons.dat в образе, закинуть туда всю папку icons,
скомпилированные @icon и iconmngr, поправить autorun.dat (ICON заменить на ICONMNGR), перекомпилировать @RB,
заменив в исходнике ICON на ICONMNGR. Вроде все :) Формат иконок *.ico - 48X48X32 bit (прозрачность поддерживается)
