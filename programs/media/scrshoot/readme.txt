  Screenshooter - программа, предназначенная для фотографирования экрана и
сохранения фотографии ввиде файла BMP.

  Возможности программы:
   * поддерживаются длинные имена файлов;
   * автоматическое сворачивание окна перед съемкой;
   * можно посмотреть на снимок перед сохранением;
   * задержка перед съемкой;
   * автоматическое сохранение файла после съемки;
   * автоматическая нумерация файлов;
   * автосъемка.

	      Длинные имена файлов.
  При сохранении файлов программа использует 70-ю функцию, следовательно
имена файлов могут содержать до 260 символов.

      Автоматическое сворачивание окна перед съемкой.
  Если включить функцию "Свернуть окно", после нажатия на кнопку
"Сделать снимок экрана" сначала свернется окно, потом будет сделан снимок.

		  Просмотр снимка.
  Если включить функцию "Показать снимок", после съемки будет показан
текущий снимок. Если нажать на кнопку "Показать снимок сейчас", будет
показан последний сделанный снимок. Чтобы закрыть просмотр, надо
нажать любую клавишу.

		     Задержка.
  Если включить функцию "Задержка", перед съемкой будет задержка, равная
введенному числу. Если число не введено, задержка будет считаться равной
нулю. 1 = 1/100 секунды, 100 = 1 секунда, 1000 = 10 секунд.

		   Автосохранение.
  Если включить функцию автосохранение, сразу после съемки файл сохранится
на жесткий диск.

		    Автонумерация.
  Если включить функцию "Автонумерация" вместо "*" в имени файла будет
вставлен текущий номер снимка, при этом необходимо задать количество
знаков в номере. Если не задать начальный номер, он будет считаться
равным нулю. Лишние знаки будут заменены нулями.
  Например, если начальный номер - 1, количество знаков - 3,
имя файла "/hd0/1/menutos/screenshoot number *.bmp",
сгенерируется имя файла "/hd0/1/menutos/screenshoot number 001.bmp".
  Если функция "Автонумерация не будет включена, значок "*"
игнорируется.

		     Автосъемка.
  Чтобы начать автосъемку, необходимо:
    1) ввести полный путь к файлу, где вместо "*" будет вставляться номер
   кадра;
    2) включить автонумерацию;
    3) включить автосохранение;
    4) установить необходимую задержку (если она нужна);
    5) отключить автосворачивание и автовосстановление;
    6) нажать кнопку "Начать автосъемку".
  Для остановки автосъемки необходимо нажать кнопку "Остановить автосъемку".

		   Горячие клавиши:
  PrintScrn - сделать снимок экрана (глобальная).
