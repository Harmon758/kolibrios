О программе

Программа для создания и редактирования 3-х мерной воксельной графики.

Внешний вид программы.

Окно программы содержит панель инструментов, текстовые подписи, буфер с изображением 3-х мерного объекта и сечение этого объекта. Через буфер с сечением объекта можно редактировать изображение. В текстовых подписях указано: текущий цвет карандаша, позиция курсора, размер кисти, масштаб и номер сечения. Изображения сохраняются в бинарный файл с расширением *.vox.

Лицензия свободная.

Изменение текущего цвета карадаша

Цвет карандаша можно поменять вызвав для этого палитру стандартных цветов кнопкой .

Также цвет карандаша можно поменять и без палитры нажав на ПКМ в одном из 2-х буферов. Цвет будет определен в зависимости от места нажатия.

Работа с кистью

Кисть может использоваться для создания изображений с повторяющимися похожими друг на друа частями. Для копирования определенной части изображения выполните такие действия:

  • откройте или нарисуйте изображение, часть которого нужно копировать
  • выберите инструмент кисть (кнопка )
  • нажмите на плоскости сечения в область, которая будет использована как образец для создания кисти
  • при необходимости измените размер области кнопками (максимальный размер кисти 32*32)
  • нажмите кнопку для запоминания выделенной области
  • выберите на плоскости сечения новую позицию для вставки запомненного образца, нажмите

Если необходимо создать несколько копий образца то последний шаг может повторятся много раз.

Если при копировании в выделенной области цвет вокселей совпадет с текущим цветом карандаша, то при изменении цвета карандаша воксели с данным цветом при вставке будут также с измененным цветом. Потому кисть также можно использовать для замены цвета.
