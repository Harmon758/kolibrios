О программе

Программа для создания 3-х мерной воксельной графики с использованием 3-х текстур.

Внешний вид программы.

Для создания 3-х мерного объекта необходимо приготовить 3 текстуры: верхнюю, боковою и лицевую. Области черного цвета на текстурах считаются прозрачными. Созданное изображение может быть сохранено в бинарный файл с расширением *.vox.

Лицензия свободная.

Работа с программой

Описание панели инструментов

- создание нового файла.

- открытие файлов с помощью диалогового окна.

- сохранение файлов.

- открытие верхней текстуры.

- открытие боковой текстуры.

- открытие лицевой текстуры.

- начать создание объекта.

- начать создание объекта с добавлением к уже существующему.

- прекратить создание объекта.

- поворот изображения.
