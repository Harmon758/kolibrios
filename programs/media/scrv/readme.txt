
=========================================

 ScrV 0.2

=========================================

 Simple program to view ZX Spectrum
 screen files.
 6912 bytes (*.scr, *.s) 
 and
 6929 bytes (*.$c) files are supported now.

 Простая программа для просмотра
 файлов экранов ZX Spectrum.
 Поддерживаются файлы размером 
 6912 байт (*.scr, *.s)
 и
 6929 байт (*.$c).
