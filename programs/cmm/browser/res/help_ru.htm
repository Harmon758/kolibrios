Справка WebView

Клавиши быстрого вызова

[CTRL + N или CTRL + T] Новое окно
[CTRL + R или F5] Перезагрузить страницу
[CTRL + O] Вызвать диалог выбора файла
[CTRL + U] Смотреть исходный код страницы
[CTRL + H] Смотреть историю посещенных страниц
[CTRL + J] Открыть загрузчик файлов
[CTRL + W] Закрыть браузер
[CTRL + ENTER] Поиск в интернете с помощью Google
[CTRL + ВЛЕВО] Аналогично кнопке "Назад" в браузере
[CTRL + ВПРАВО] Аналогично кнопке "Вперед"
[F6] Перемещает текстовый курсор в омнибокс
