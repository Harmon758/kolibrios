WebView Help


Shortcut keys
 
[CTRL + N or CTRL + T] New window
[CTRL + R or F5] Refresh the current page
[CTRL + U] View a web page's source code
[CTRL + H] Open browser history
[CTRL + J] Display the file downloader
[CTRL + W] Exit browser
[CTRL + ENTER] Submit a web search
[CTRL + LEFT] Back a page.
[CTRL + RIGHT] Forward a page.
[F6] Moves a text cursor to the omnibox
