Введение

Библиотека box_lib.obj содержит разные элементы управления, которые можно использовать при написании приложений. Файл box_lib.mac подключает макросы для компиляции библиотеки и для использования элементов.

Элементы библиотеки

  • editbox
  • checkbox
  • optionbox
  • scrollbar
  • d_button
  • menubar
  • filebrowser
  • tree_list
