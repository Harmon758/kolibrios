Info 3ds

Info3ds - программа просмотра структуры файлов формата *.3ds. Все файлы *.3ds состоят из определенных блоков, в которых находится информация о объектах сцены. В программе слева в виде дерева показана структура блоков в файле, а справа выбранный блок показывается в шестнадцатеричном виде. Для просмотра этих файлов в 3-х мерном виде используйте программу View3ds.

Рис. 1. Внешний вид программы

Координаты вершин объекта можно просматривать в виде дробных чисел. Для просмотра координат нужно (см. рис. 2):

1) веделить блок с кодом 4110 (в русской версии подписан как Список вершин),

2) нажать кнопку на панели инструментов.
